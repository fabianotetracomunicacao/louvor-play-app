Protótipo isolado · lógica inline

Editar o visual final sem abandonar o código atual

Aqui cada linha musical já nasce no formato que o sistema entende por trás: texto com acordes inline. O editor visual só renderiza essa mesma estrutura de forma mais natural, para encaixar com importação externa, serialização e o renderer atual.

Como pensar esse editor

  • A base da linha continua sendo o código inline do sistema.
  • Importação externa entra no pipeline atual, vira código, e este editor lê o código.
  • Ao editar visualmente, o protótipo reserializa para o mesmo formato inline.
  • Os acordes agora são âncoras da própria linha, não uma linha separada.

Este painel representa o código que precisa encaixar com o sistema atual. Você também pode colar código existente aqui e clicar em “Ler código”.

Atalhos importantes

  • Enter em linha musical cria uma nova linha abaixo.
  • Clique sobre um acorde para editar ou remover.
  • Use “Inserir acorde” com o cursor na linha para ancorar o acorde no ponto certo.
